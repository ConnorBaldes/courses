### Course Summaries:

#### **CS160: Introduction to Computer Science** 
![Python Language Stats](https://img.shields.io/badge/Python-100.0%25-brightgreen)

Introduction to Computer Science covers foundational programming concepts, focusing on problem-solving using computational methods. The course introduces basic programming techniques, including control structures, functions, and data types. It also covers fundamental algorithms and data structures such as arrays, lists, and basic sorting algorithms, alongside topics in object-oriented programming, debugging, and software development processes.

#### **CS161: Introduction to Computer Science I**
![C++ Language Stats](https://img.shields.io/badge/C++-100.0%25-brightgreen)

Introduction to Computer Science II builds upon the concepts learned in CS160 and introduces more advanced topics in computer science. The course covers data structures such as stacks, queues, trees, and graphs, and explores algorithms for searching and sorting these structures. It delves into recursion, dynamic memory allocation, and more advanced techniques in object-oriented programming.

Introduction to Computer Science III builds on the knowledge from CS160 and CS161, focusing on advanced data structures and algorithms. The course covers topics such as linked lists, hash tables, advanced tree structures, and graph algorithms. It introduces more complex algorithmic techniques, including divide and conquer, dynamic programming, and greedy algorithms. The course also emphasizes memory management, performance analysis, and software design principles.

#### **CS261: Data Structures**
![C Language Stats](https://img.shields.io/badge/C-98.26%25-brightgreen)
![Makefile Language Stats](https://img.shields.io/badge/Makefile-1.74%25-lightgray)

Data Structures focuses on the design, implementation, and analysis of fundamental data structures and algorithms. The course covers topics such as arrays, linked lists, stacks, queues, hash tables, trees, and graphs, along with their corresponding algorithms for searching, sorting, and manipulation. Emphasis is placed on understanding the performance and efficiency of different data structures and algorithms, including time and space complexity analysis.

#### **CS271: Computer Architecture and Assembly Language**
![Assembly Language Stats](https://img.shields.io/badge/Assembly-83.2%25-yellowgreen)
![C++ Language Stats](https://img.shields.io/badge/C++-14.16%25-orange)
![C Language Stats](https://img.shields.io/badge/C-2.64%25-lightgray)

Computer Architecture and Assembly Language explores computer organization and low-level programming. The course covers topics such as data representation, instruction sets, memory hierarchy, and the fundamentals of CPU design. It introduces assembly language programming, focusing on converting high-level programming concepts into machine code. The course also delves into stack management, performance optimization, and the interaction between hardware and software. Key concepts include understanding the execution flow at the hardware level and the impact of hardware architecture on software performance.

#### **CS325: Analysis of Algorithms**
![C++ Language Stats](https://img.shields.io/badge/C++-100.0%25-brightgreen)

Analysis of Algorithms focuses on the design and analysis of algorithms, with an emphasis on efficiency and problem-solving techniques. The course covers key algorithmic paradigms such as divide and conquer, dynamic programming, greedy algorithms, and graph algorithms. Topics include the analysis of time and space complexity using Big-O notation, algorithm optimization, and the trade-offs between different approaches. The course also explores NP-completeness and the limits of algorithmic solvability.

#### **CS331: Introduction to Artificial Intelligence**
![Python Language Stats](https://img.shields.io/badge/Python-82.4%25-yellowgreen)
![Shell Language Stats](https://img.shields.io/badge/Shell-17.6%25-orange)

Introduction to Artificial Intelligence provides an overview of the fundamental concepts and techniques in AI. The course covers topics such as search algorithms, knowledge representation, reasoning, and machine learning. Key areas of focus include informed and uninformed search strategies, decision trees, neural networks, and natural language processing. The course also explores problem-solving in AI, including constraint satisfaction, game playing, and optimization techniques. 

#### **CS340: Introduction to Databases**
![Jinja Language Stats](https://img.shields.io/badge/Jinja-58.12%25-yellow)
![Python Language Stats](https://img.shields.io/badge/Python-41.88%25-yellow)

 Introduction to Databases covers the fundamental concepts of database design, implementation, and management. The course introduces relational database theory, SQL for querying and manipulating data, and the structure of database management systems (DBMS). Topics include entity-relationship modeling, normalization, indexing, and transactions. The course also explores database security, concurrency control, and optimization techniques. Practical assignments focus on designing databases, writing complex SQL queries, and developing database-driven applications.

#### **CS362: Software Engineering II**
![Python Language Stats](https://img.shields.io/badge/Python-100.0%25-brightgreen)

Software Engineering II focuses on advanced software engineering principles and practices, emphasizing the design, development, and management of large-scale software projects. The course covers topics such as software architecture, design patterns, testing strategies, and version control. It explores methodologies for ensuring software quality, including unit testing, integration testing, and debugging techniques. The course also delves into project management practices, such as agile methodologies, team collaboration, and effective communication.

#### **CS381: Programming Language Fundamentals**
![Haskell Language Stats](https://img.shields.io/badge/Haskell-90.23%25-brightgreen)
![Prolog Language Stats](https://img.shields.io/badge/Prolog-9.77%25-lightgray)

Programming Language Fundamentals explores the theory and design of programming languages. The course covers key concepts such as syntax, semantics, and the implementation of different language paradigms, including procedural, functional, and object-oriented programming. Topics include language parsing, type systems, memory management, and the principles behind language interpreters and compilers. The course also examines how programming languages handle variables, control structures, and data types, with a focus on understanding how language features affect program behavior and performance.

#### **CS434: Machine Learning & Data Mining**
![Python Language Stats](https://img.shields.io/badge/Python-100.0%25-brightgreen)

Machine Learning & Data Mining covers the fundamental concepts and techniques used in analyzing and extracting patterns from large datasets. The course introduces key topics in supervised and unsupervised learning, including classification, regression, clustering, and dimensionality reduction. It explores algorithms such as decision trees, support vector machines, k-means, and neural networks. The course also covers data preprocessing, feature selection, and model evaluation techniques.

#### **CS492: Mobile Software Development**
![Kotlin Language Stats](https://img.shields.io/badge/Kotlin-100.0%25-brightgreen)

Mobile Software Development focuses on the design and development of mobile applications for platforms such as Android and iOS.
